Why coding?

The reason I chose to become a full-stack developer is because I think that coding is an excellent tool to not only learn about programming and computers but also a way for a person to discover how the surrounding things really work. By learning how to write code, we also learn how to improve problem-solving skills, which are very usefull skils in our lives. Computer programming is extremely creative field. By experimenting with our code we get the chance to do something on our own.
